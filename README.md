# Todo App

## About

This is a simple Todo App built using **HTML**, **CSS**, and **Vanilla JavaScript**.

The app allows users to add tasks, edit them, change their status, and delete them. All tasks are saved in the browser using **Local Storage**, so they are still available after refreshing the page.

This project was created to practice JavaScript fundamentals, DOM manipulation, events, arrays, objects, and Local Storage.

---

## Features

* Add a new task
* Edit an existing task
* Mark a task as **Active**
* Mark a task as **Completed**
* Delete a task
* Automatically save tasks in Local Storage


---

## Technologies Used

* HTML5
* CSS3

## How It Works

1. Enter a task in the input field.
2. Click the **Add Task** button.
3. The task is saved in Local Storage.
4. All saved tasks are displayed in the table.
5. You can:

   * Edit the task
   * Change its status to Active
   * Change its status to Completed
   * Delete the task

Every time a change is made, the table updates automatically.
